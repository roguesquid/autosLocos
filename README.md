## Descripción general

El juego consiste en definir una serie de vehículos que competirán en una carrera simulada, el objetivo es determinar qué vehículo es el ganador, mediante una apuesta. Esta carrera se simulará con símbolos que se moverán en la pantalla.

### El reto

Los usuarios deben ser capaces de:

- Utilizar la aplicacion de simulacion de carreras con los siguientes requerimientos:
  - Configurar la carrera
  - Indicar cantidad de carriles
  - Generar carriles
  - Seleccionar vehiculos
  - Comenzar la carrera
  - Efecto de los obstaculos
  - Guardar tabla de posiciones por cada carrera
- Realizar operaciones sobre los vehiculos:
  - Agregar
  - Modificar
  - Eliminar
  - Consultar
  - Listar
